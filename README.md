# 🎮Flappy-Bird

## ℹAbout:
This is a desktop game, a clone of the flappy bird mobile game.

## 💻To deploy localy:
```
1)git clone https://github.com/iFoxtrot33/Flappy-Bird.git
2)cd Doodle-Jump
3)make install
```
